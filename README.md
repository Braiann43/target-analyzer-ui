# 🎯 Target Analyzer - Web Scraper Frontend

Este proyecto es el esqueleto del Front-End para una futura aplicación de Web Scrapping. La interfaz está diseñada con un estilo de terminal/hacker (Cyberpunk) responsivo, pensado para mostrar los datos extraídos de una URL objetivo.

## 🚀 Tecnologías Utilizadas

* **HTML5:** Estructura semántica del panel de control.
* **CSS3:** Variables globales, CSS Grid, Flexbox y animaciones. No se utilizaron frameworks externos.

## 🛠️ Estructura de la Interfaz

La interfaz, denominada "Bunker", se divide en una consola de entrada y una grilla táctica de 4 paneles:
1.  **VISTA:** Para renderizar capturas o estados del sitio objetivo.
2.  **TECNOLOGÍAS:** Para listar el stack tecnológico detectado.
3.  **ENLACES:** Panel con scroll interno para listar los links encontrados.
4.  **MÉTRICAS:** Tiempos de respuesta, peso de la página, etc.
